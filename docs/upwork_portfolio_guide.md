# Upwork Portfolio Upload Guide
## Project: Nexoria Commerce Inc. — Finance Automation & Analytics Suite

This guide explains how to fill out the **Add a New Portfolio Project** form on Upwork based on the updated single-page layout and character limits.

---

## 🚶 Step-by-Step Portfolio Upload Form

### 1. Project Title (Max 70 Characters)
Copy and paste this title into the **Project title** box (57 characters):
```text
Financial Automation & BI Suite: Excel, Python & Power BI
```

### 2. Your Role (Optional - Max 100 Characters)
Copy and paste this role description (48 characters):
```text
Financial Systems Analyst & Automation Developer
```

### 3. Project Description (Max 600 Characters)
Copy and paste this high-impact, metric-focused summary (523 characters):
```text
Automated end-to-end FP&A and billing workflows for Nexoria Commerce, a B2B distributor:
- Automated Billing: Set up program-driven PDF invoicing via Python & Excel VBA (saved 40+ hours/week).
- Tax Compliance: Power Query lookups with category exemptions (saved $8,000/year).
- Freight Reconciler: Weighted cost allocation algorithm, boosting margin accuracy from 78% to 97%+.
- Executive Dashboard: 2-page Power BI dashboard for budget vs. actual variance analysis.
- P&L & AR reports: Custom spreadsheets with COM automation and AR aging warning buckets.
```

### 4. Add Content (Dashed Upload Panel on Right)
Use the interactive icons in the dashed panel to upload your project assets:
* **Add Link (🔗 Icon)**: Add your public GitHub repository link: 
  `https://github.com/KSensei99/finance-analyst-portfolio`
* **Add Document (📄 Icon)**: Upload the exported PDF layout:
  `04_budget_vs_actual/Nexoria_BVA_Dashboard.pdf`
* **Add Image (🖼️ Icon)**: Upload screenshots of the Power BI canvas, the P&L month dropdown, or the forecasting chart (`revenue_forecast.png`).

### 5. Skills and Deliverables (Max 5 Skills)
Type and add these exact 5 skills to match the limit:
1. `Excel VBA`
2. `Power BI`
3. `Financial Planning`
4. `Python`
5. `Power Query`

## 🚀 Publish
Scroll down to the bottom of the page and click **Save** (or **Publish**) to add this item to your Upwork profile.
